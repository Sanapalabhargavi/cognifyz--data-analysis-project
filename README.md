# 🍽️ Cognifyz Data Analysis Internship Project

## 📊 Project Overview

This project analyzes restaurant data to extract insights on cuisines, ratings, pricing, and services.

## 🔹 Tasks Completed

### Level 1

* Top Cuisines Analysis
* City-wise Restaurant Analysis
* Price Range Distribution
* Online Delivery Analysis

### Level 2

* Ratings Distribution
* Cuisine Combination Analysis
* Geographic Analysis
* Restaurant Chain Analysis

### Level 3

* Votes Analysis
* Price vs Services Analysis

## 📈 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📌 Key Insights

* Popular cuisines and cities identified
* Higher votes often correlate with better ratings
* Expensive restaurants tend to offer more services
